# Base Galaga - 게임 스토리

## 🌌 세계관

### 배경 이야기

서기 2084년, 인류는 마침내 Base 네트워크를 통해 우주 탐사의 새로운 시대를 열었습니다. 하지만 평화는 오래가지 않았습니다. 알 수 없는 외계 함대가 Base 네트워크의 노드들을 하나씩 공격하기 시작했습니다.

당신은 Base Defense Force의 엘리트 파일럿입니다. 최신예 전투기를 타고 10개의 Base 노드를 방어해야 합니다. 각 노드는 점점 더 강력한 적들로 가득 차 있으며, 5번째와 10번째 노드에는 거대한 모선이 대기하고 있습니다.

Farcaster 네트워크를 통해 전 우주의 파일럿들과 연결되어 있습니다. 당신의 전투 기록은 실시간으로 공유되며, 최고의 파일럿만이 리더보드에 이름을 올릴 수 있습니다.

**미션**: 10개의 Base 노드를 모두 방어하고 인류를 구하세요!

## 🎮 게임 플레이 흐름

### 1. 미션 시작 (Main Menu)

게임을 시작하면 메인 메뉴가 나타납니다. 화면 상단에는 당신의 Farcaster 아이덴티티가 표시됩니다 - 당신의 콜사인(username)과 파일럿 번호(FID)입니다.

**"INITIATE LAUNCH"** 버튼을 누르면 첫 번째 미션이 시작됩니다.

### 2. 전투 시작 (Game Scene)

#### 첫 번째 노드 (Stage 1)

화면이 어두워지고 깊은 우주가 펼쳐집니다. 별들이 반짝이고 먼 곳에서 네뷸라가 빛납니다. 당신의 전투기가 화면 하단에 나타나며, 곧 적들이 포메이션을 이루며 접근합니다.

**적의 움직임:**
- 적들은 그리드 포메이션으로 정렬되어 있습니다
- 천천히 좌우로 움직이며 하강합니다
- 가끔 당신을 향해 레이저를 발사합니다

**당신의 임무:**
- 화살표 키로 전투기를 조종합니다
- 스페이스바를 눌러 레이저를 발사합니다
- 모든 적을 격파하세요!

#### 스테이지 클리어와 파워업

모든 적을 격파하면 잠시 전투가 멈춥니다. 화면에 3개의 파워업 옵션이 나타납니다:

**선택의 순간:**
- **실드 (Shield)**: 10초간 무적 상태
- **속도 부스트 (Speed Boost)**: 15초간 이동 속도 증가
- **연사력 증가 (Fire Rate)**: 15초간 발사 속도 2배
- **체력 회복 (Health)**: HP 1 회복
- **최대 체력 증가 (Max Health)**: 영구적으로 최대 HP +1
- **무기 업그레이드 (Weapon Upgrade)**: 무기를 다음 단계로 업그레이드

**전략적 선택:**
- 체력이 낮다면? → Health 선택
- 다음 스테이지가 어렵다면? → Shield나 Weapon Upgrade
- 보스전이 다가온다면? → 무기 업그레이드 추천!

#### 무기 진화 시스템

당신의 전투기는 5단계의 무기 시스템을 가지고 있습니다:

1. **SINGLE SHOT** (기본)
   - 단일 레이저 발사
   - 정확하지만 화력이 약함

2. **DOUBLE SHOT** (1단계 업그레이드)
   - 2발의 레이저를 동시 발사
   - 명중률 향상

3. **TRIPLE SHOT** (2단계 업그레이드)
   - 3발의 레이저를 부채꼴로 발사
   - 넓은 범위 커버

4. **SPREAD SHOT** (3단계 업그레이드)
   - 5발의 레이저를 광범위하게 발사
   - 화면을 뒤덮는 탄막

5. **LASER BEAM** (최종 업그레이드)
   - 연속 레이저 빔 발사
   - 최강의 화력, 하지만 정확한 조준 필요

### 3. 난이도 상승

#### Stage 2-4: 점점 강해지는 적들

스테이지가 올라갈수록:
- 적의 수가 증가합니다 (10마리 → 12마리 → 15마리...)
- 적의 이동 속도가 빨라집니다
- 적의 체력이 증가합니다
- 적의 공격 빈도가 높아집니다

**새로운 포메이션:**
- **V Formation**: V자 형태로 날아오는 적들
- **Circle Formation**: 원형으로 회전하며 공격
- **Wave Formation**: 파도처럼 물결치며 접근

### 4. 첫 번째 보스전 (Stage 5)

**경고: BOSS APPROACHING**

거대한 모선이 화면 상단에 나타납니다. 일반 적들보다 10배 이상 큰 크기에 체력 바가 화면에 표시됩니다.

**보스의 공격 패턴:**

**Phase 1 (HP 100% - 66%):**
- 좌우로 천천히 이동
- 3발씩 묶음으로 레이저 발사
- 예측 가능한 패턴

**Phase 2 (HP 66% - 33%):**
- 이동 속도 증가
- 5발의 부채꼴 레이저
- 가끔 플레이어를 향해 조준 사격

**Phase 3 (HP 33% - 0%):**
- 빠른 좌우 이동
- 나선형 탄막 패턴
- 플레이어 위치 예측 공격
- 급강하 공격 추가

**보스 격파:**
보스를 격파하면 대량의 점수와 함께 다음 스테이지로 진행합니다. 이제 당신은 절반의 노드를 방어했습니다!

### 5. Stage 6-9: 최종 결전 준비

보스를 격파한 후의 스테이지들은 더욱 가혹합니다:
- 적들이 더 공격적으로 변합니다
- 탄막의 밀도가 높아집니다
- 새로운 적 타입이 등장합니다

**생존 전략:**
- 파워업을 현명하게 선택하세요
- 무기를 최대한 업그레이드하세요
- 체력을 항상 체크하세요
- 실드는 위급할 때를 위해 아껴두세요

### 6. 최종 보스전 (Stage 10)

**FINAL BOSS: MOTHERSHIP**

마지막 노드를 지키는 것은 외계 함대의 총사령관 모선입니다. 이전 보스보다 2배 강력하며, 완전히 다른 공격 패턴을 가지고 있습니다.

**최종 보스의 특징:**
- 체력이 이전 보스의 2배
- 4가지 공격 패턴을 무작위로 사용
- 화면 전체를 커버하는 탄막
- 예측 불가능한 움직임

**승리의 순간:**
최종 보스를 격파하면 **"MISSION ACCOMPLISHED"** 메시지와 함께 게임이 클리어됩니다. 당신은 모든 Base 노드를 방어하고 인류를 구했습니다!

### 7. 미션 리포트 (Game Over)

전투가 끝나면 미션 리포트 화면이 나타납니다:

**표시 정보:**
- **FINAL SCORE**: 최종 점수
  - 기본 점수 (적 격파)
  - 시간 보너스 (빠른 클리어)
  - 스테이지 보너스
- **STAGE**: 도달한 스테이지
- **TIME**: 총 플레이 시간
- **NEW RECORD**: 최고 기록 갱신 시 표시

**선택 옵션:**
- **RE-DEPLOY**: 다시 도전
- **BROADCAST SCORE**: Farcaster에 점수 공유
- **LEADERBOARD**: 리더보드 확인
- **ABORT TO MENU**: 메인 메뉴로

## 🎯 점수 시스템 상세

### 점수 계산 방식

```
최종 점수 = 기본 점수 + 시간 보너스 + 스테이지 보너스
```

**기본 점수:**
- 일반 적 격파: 100점
- 보스 격파: 5000점

**시간 보너스:**
- 5분(300초) 이내 클리어 시 보너스
- 공식: (300 - 실제 시간) × 1000점
- 예: 4분 클리어 = 60,000점 보너스

**스테이지 보너스:**
- 각 스테이지 × 1000점
- Stage 10 클리어 = 10,000점 보너스

**예시:**
- 기본 점수: 50,000점 (적 격파)
- 시간 보너스: 60,000점 (4분 클리어)
- 스테이지 보너스: 10,000점 (Stage 10)
- **총점: 120,000점**

## 🕹️ 조작 방법

### PC (키보드)
- **↑ ↓ ← →**: 전투기 이동
- **Space**: 레이저 발사
- **Shift**: 스킬 발동 (쿨타임 있음)

### 모바일 (터치)
- **가상 D-Pad**: 전투기 이동
- **FIRE 버튼**: 자동 연사
- **SKILL 버튼**: 스킬 발동

## 🌟 전략 가이드

### 초보자를 위한 팁

1. **움직임이 생명**
   - 한 곳에 가만히 있지 마세요
   - 적의 레이저를 예측하고 피하세요

2. **무기 업그레이드 우선**
   - 초반에는 무기 업그레이드에 집중
   - LASER까지 업그레이드하면 훨씬 쉬워집니다

3. **체력 관리**
   - 체력이 1이 되기 전에 회복하세요
   - 보스전 전에는 체력을 최대로 유지

4. **실드 타이밍**
   - 보스의 Phase 3에서 사용하면 효과적
   - 탄막이 너무 많을 때 긴급 사용

### 고급 전략

1. **적 패턴 암기**
   - 각 스테이지의 적 포메이션을 외우세요
   - 어느 적이 먼저 공격하는지 파악

2. **점수 최적화**
   - 빠른 클리어로 시간 보너스 획득
   - 모든 적을 격파하여 기본 점수 극대화

3. **파워업 조합**
   - Speed Boost + Fire Rate = 최강 조합
   - Shield + Weapon Upgrade = 보스전 필승
